# Netflix Data Analysis

## Project Description

This project analyzes a subset of the Netflix dataset using Python. The goal is to perform data cleaning and data visualization to understand patterns and trends in Netflix content.

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* VS Code

## Steps Performed

1. Loaded the Netflix dataset.
2. Cleaned the dataset by handling missing values and removing duplicates.
3. Processed and analyzed the data using Pandas.
4. Created visualizations using Matplotlib to explore trends and patterns.

## Key Insights

* Distribution of movies and TV shows on Netflix.
* Trends in content release years.
* Basic insights from visual analysis.
